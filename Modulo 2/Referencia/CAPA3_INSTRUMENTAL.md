## 1. QUÉ ES Y QUÉ NO ES — VISIÓN INSTRUMENTAL

### CAPA 3 ES:
- El conjunto de herramientas para **acceder al N3** (sistemas emocionales) y **leer** sus patrones
- Tres canales de comunicación que hablan el lenguaje del N3 (sensorial/emocional/somático)
- Un sistema de lectura y mapeo — no de interpretación ni de imposición

### CAPA 3 NO ES:
- Adivinación, intuición no verificable ni lectura subjetiva
- Un método que requiere "don" o capacidad especial
- Interpretación del terapeuta sobre lo que "debería" sentir el paciente

### PRINCIPIO RECTOR:
Cada herramienta accede a un aspecto diferente de la impronta: **cuándo** se instaló (test muscular), **dónde** vive en el cuerpo (localización somática), **cómo** se siente (rastreo sensorial). La integración de los 3 canales produce una cartografía completa.

## 2. TEST MUSCULAR — CANAL NEUROMUSCULAR

### 2.1 Definición

El test muscular es una **respuesta ideomotora neuromuscular**. No es una medición de fuerza ni un acto de adivinación.

| ATRIBUTO | VALOR |
|---|---|
| **Naturaleza** | Respuesta ideomotora neuromuscular |
| **Base** | Modulación ideomotora — el sistema nervioso ajusta el tono muscular según el contenido informacional |
| **Función** | Canal de comunicación con el N3 (campo bioeléctrico) que bypasea el N4 (cognición verbal) |
| **Formato** | Binario: sí/no, congruente/incongruente |
| **Equivalente TAME** | Análogo a electrodos/voltage dyes de Levin para leer el campo bioeléctrico |

### 2.2 Mecanismo

```
ESTÍMULO INFORMACIONAL (pregunta, frase, imagen)
        ↓
PROCESAMIENTO N3 (campo bioeléctrico evalúa congruencia)
        ↓
MODULACIÓN IDEOMOTORA (N3 ajusta tono muscular)
        ↓
RESPUESTA MUSCULAR (fuerte = congruente / débil = incongruente)
```

**Clave:** El N3 responde ANTES de que el N4 pueda procesar verbalmente. Por eso el test muscular accede a información que el paciente no puede reportar conscientemente.

### 2.3 Secuencia de Demostración (4 pasos)

| PASO | NOMBRE | PROCEDIMIENTO | RESULTADO ESPERADO |
|---|---|---|---|
| 1 | **Calibración** | Establecer presión base. Terapeuta presiona brazo del paciente con fuerza constante | Se establece referencia de tono muscular basal |
| 2 | **Test Verdadero** | Paciente dice su nombre real mientras se testea | Músculo mantiene tono (congruencia informacional) |
| 3 | **Test Falso** | Paciente dice nombre falso mientras se testea | Músculo pierde tono (incongruencia informacional) |
| 4 | **Test Emocional** | Se introduce tema con carga emocional mientras se testea | Músculo responde según la carga del N3 sobre ese tema |

### 2.4 Protocolo de Práctica

**Duración:** 20 minutos en parejas

| FASE | DURACIÓN | ACTIVIDAD |
|---|---|---|
| 1 — Calibración mutua | 5 min | Ambos participantes se calibran mutuamente para encontrar presión base |
| 2 — Verdadero/Falso | 5 min | Alternar nombre real / nombre falso para verificar respuesta binaria |
| 3 — Primera exploración | 10 min | Introducir temas con carga emocional progresiva y observar respuesta |

### 2.5 Cuatro Errores Comunes que Invalidan el Test

| # | ERROR | POR QUÉ INVALIDA | CORRECCIÓN |
|---|---|---|---|
| 1 | **Presión excesiva** | Convierte test informacional en prueba de fuerza bruta | Usar presión constante, suave, suficiente para detectar cambio de tono |
| 2 | **Anticipación del terapeuta** | El terapeuta proyecta expectativa y modifica su propia presión inconscientemente | Mantener neutralidad, no anticipar resultado |
| 3 | **Paciente activa N4** | El paciente racionaliza o "intenta" responder correctamente | Instruir al paciente para que no intente controlar la respuesta |
| 4 | **Preguntas ambiguas** | El N3 responde literalmente — una pregunta ambigua genera respuesta ambigua | Formular preguntas claras, específicas, binarias |

### 2.6 Conexión TAME

```
LEVIN usa:                          BV4 usa:
─────────                           ─────────
Electrodos para leer Vmem       →   Test muscular para leer
                                     respuesta del campo bioeléctrico

Voltage dyes para visualizar    →   Respuesta ideomotora para
patrones bioeléctricos               detectar congruencia/incongruencia

Medición objetiva del campo     →   Medición funcional del campo
eléctrico celular                    bioeléctrico colectivo
```

## 3. CANAL SOMÁTICO-SENSORIAL — LOCALIZACIÓN CORPORAL

### 3.1 Definición

El canal somático-sensorial localiza **dónde** vive la impronta en el campo corporal. Identifica la zona de aislamiento bioeléctrico asociada al patrón de supervivencia.

| ATRIBUTO | VALOR |
|---|---|
| **Función** | Localización espacial del patrón en el campo corporal |
| **Accede a** | Zona de aislamiento bioeléctrico (Capa 2) |
| **Formato** | Ubicación anatómica + cualidades sensoriales |
| **Lenguaje** | Sensaciones corporales localizadas |

### 3.2 Lógica Corporal

Las improntas no se almacenan al azar en el cuerpo. Existe una lógica corporal:

```
ZONA CORPORAL              ASOCIACIÓN FUNCIONAL
────────────               ────────────────────
Garganta/cuello        →   Expresión bloqueada, lo no dicho
Pecho/corazón          →   Vínculos, pérdida, amor
Plexo solar            →   Poder, control, identidad
Abdomen/intestino      →   Supervivencia básica, miedo visceral
Pelvis                 →   Sexualidad, creatividad, identidad sexual
Espalda                →   Carga, responsabilidad, soporte
Hombros                →   Peso del mundo, obligaciones
```

**Nota:** Esta es una guía orientativa. La localización específica se CONFIRMA siempre con el test muscular (Canal 1). El terapeuta NO asigna — el sistema del paciente INDICA.

### 3.3 Procedimiento

1. **Pregunta al paciente:** "¿Dónde sientes esto en tu cuerpo?"
2. **Confirmación con test muscular:** Se verifica la zona reportada
3. **Precisión:** Si la zona es amplia, se reduce por bisección (arriba/abajo, izquierda/derecha)
4. **Registro:** Se documenta zona exacta para la cartografía

## 4. CANAL EMOCIONAL-ASOCIATIVO — RASTREO DE SENSACIONES VISCERALES

### 4.1 Definición

El canal emocional-asociativo rastrea la **cadena de sensaciones** que el N3 produce cuando se activa una impronta. No trabaja con emociones nombradas sino con cualidades sensoriales directas.

| ATRIBUTO | VALOR |
|---|---|
| **Función** | Describir CÓMO se siente la impronta en el cuerpo |
| **Accede a** | Cualidades sensoriales del patrón activado |
| **Formato** | Cadena de sensaciones: forma → peso → temperatura → movimiento → color |
| **Lenguaje** | Sensaciones viscerales directas (no nombres de emociones) |

### 4.2 Cadena de Sensaciones (5 dimensiones)

| DIMENSIÓN | PREGUNTA | EJEMPLOS |
|---|---|---|
| **Forma** | ¿Qué forma tiene? | Nudo, bola, presión, vacío, punta, bloque |
| **Peso** | ¿Cuánto pesa? | Ligero, pesado, aplastante, flotante |
| **Temperatura** | ¿Qué temperatura tiene? | Frío, caliente, helado, ardiente, tibio |
| **Movimiento** | ¿Se mueve? ¿Hacia dónde? | Expansivo, contraído, pulsante, estático, ascendente |
| **Color** | ¿Qué color tiene? | Negro, rojo, gris, oscuro, brillante |

### 4.3 Las 20 Sensaciones Viscerales

El Módulo 2 (Improntas de Supervivencia) enseña 20 sensaciones viscerales específicas que el N3 utiliza como lenguaje. Estas se clasifican por sistema corporal y representan el vocabulario somático del N3.

**Referencia:** Ver Capa 5 (Clínica) para la lista completa de las 20 sensaciones y su asociación con cada impronta.

### 4.4 Distinción Clave: Sensación vs. Emoción Nombrada

```
EVITAR (lenguaje N4):        USAR (lenguaje N3):
─────────────────            ───────────────────
"Siento tristeza"        →   "Hay un peso frío en el pecho"
"Tengo miedo"            →   "Hay una contracción helada en el abdomen"
"Estoy enojado"          →   "Hay algo caliente y expansivo en el plexo"
"Me siento abandonado"   →   "Hay un vacío frío en el pecho izquierdo"

La emoción NOMBRADA es N4. La sensación DESCRITA es N3.
El canal 3 trabaja con descripciones, no con etiquetas.
```

## 5. PROTOCOLO BÁSICO DE EXPLORACIÓN — INTEGRACIÓN DE 3 CANALES

### 5.1 Secuencia de 3 Pasos

| PASO | CANAL | PREGUNTA CLAVE | DATO QUE APORTA |
|---|---|---|---|
| 1 | **Neuromuscular** (Test Muscular) | ¿Hay un patrón activo sobre este tema? | CUÁNDO se instaló, SI está activo |
| 2 | **Somático-Sensorial** (Localización) | ¿Dónde lo sientes en el cuerpo? | DÓNDE vive la impronta |
| 3 | **Emocional-Asociativo** (Rastreo) | ¿Qué forma, peso, temperatura tiene? | CÓMO se siente la impronta |

### 5.2 Diagrama de Integración

```
         CANAL 1                    CANAL 2                    CANAL 3
    (Neuromuscular)          (Somático-Sensorial)        (Emocional-Asociativo)
          │                         │                          │
    ¿Hay patrón?              ¿Dónde está?              ¿Cómo se siente?
    ¿Cuándo se instaló?      ¿En qué zona?             ¿Qué forma tiene?
          │                         │                          │
    Respuesta binaria         Ubicación corporal          Cadena sensorial
    (sí/no, edad, persona)    (zona específica)           (forma/peso/temp/mov/color)
          │                         │                          │
          └─────────────────────────┼──────────────────────────┘
                                    │
                           CARTOGRAFÍA INTEGRADA
                    (mapa completo de la impronta)
```

### 5.3 Lógica de Integración

Cada canal accede a un **aspecto diferente** de la misma impronta:

- **Canal 1** → Dimensión TEMPORAL (cuándo se grabó, quién estuvo involucrado)
- **Canal 2** → Dimensión ESPACIAL (dónde se almacena en el campo corporal)
- **Canal 3** → Dimensión CUALITATIVA (cómo se manifiesta sensorialmente)

**Analogía TAME:** Así como Levin necesita múltiples instrumentos para entender el estado bioeléctrico de un organismo (electrodos para voltaje, dyes para visualizar patrones, registros temporales para cambios), BV4 necesita 3 canales para mapear completamente una impronta.

## 6. CARTOGRAFÍA — MAPA COMPLETO DE LA IMPRONTA

### 6.1 Definición

La cartografía es el **mapa completo** de un patrón de supervivencia. Integra los datos de los 3 canales para producir una imagen detallada de la impronta.

| ATRIBUTO | VALOR |
|---|---|
| **Naturaleza** | LECTURA del sistema, no interpretación del terapeuta |
| **Contenido** | Edad de origen, personas involucradas, emoción central, creencia, zona corporal, manifestación actual |
| **Principio** | El sistema del paciente muestra su estructura — el terapeuta no adivina ni impone |
| **Resultado** | Coordenadas precisas para el reprocesamiento (Capa 4: Context Engineering) |

### 6.2 Cartografía = Lectura, NO Interpretación

```
LECTURA (correcto):                  INTERPRETACIÓN (incorrecto):
──────────────────                   ────────────────────────────
"El test indica edad 4"          →   "Probablemente fue a los 4"
"El cuerpo señala pecho izq"     →   "Yo creo que es el corazón"
"La sensación es nudo frío"      →   "Eso suena a abandono"
"Test confirma figura materna"   →   "Seguramente fue tu madre"

El terapeuta PREGUNTA y REGISTRA.
El sistema del paciente RESPONDE y CONFIRMA.
```

### 6.3 Tracking Temporal por Bisección

Técnica para localizar la edad de origen de la impronta con precisión:

```
EJEMPLO DE BISECCIÓN TEMPORAL:

Pregunta: "¿Este patrón se originó antes de los 20 años?"
Test: SÍ
Pregunta: "¿Antes de los 10?"
Test: SÍ
Pregunta: "¿Antes de los 5?"
Test: SÍ
Pregunta: "¿Antes de los 3?"
Test: NO
Pregunta: "¿A los 3?"
Test: NO
Pregunta: "¿A los 4?"
Test: SÍ
→ RESULTADO: Origen a los 4 años

Rango de división: 20 → 10 → 5 → 3 → 4 (5 preguntas para localizar edad exacta)
```

### 6.4 Protocolo de Práctica — Mini-Cartografía

| PASO | ACTIVIDAD | CANAL USADO |
|---|---|---|
| 1 | **Elegir tema menor** (no traumático para práctica) | — |
| 2 | **Tracking temporal** (bisección para localizar edad) | Canal 1 (Test muscular) |
| 3 | **Identificar persona** (¿quién estaba involucrado?) | Canal 1 (Test muscular) |
| 4 | **Localización corporal** (¿dónde se siente?) | Canal 2 (Somático-sensorial) |

**Resultado esperado:** Mini-cartografía con 3 datos:

| DATO | EJEMPLO |
|---|---|
| **Edad aproximada** | 4 años |
| **Persona involucrada** | Figura materna |
| **Zona corporal** | Pecho izquierdo |

### 6.6 Tarea Inter-Sesión

| TAREA | INSTRUCCIÓN |
|---|---|
| **Observar** | Notar reacciones desproporcionadas durante la semana |
| **Registrar ubicación** | ¿Dónde en el cuerpo se siente la reacción? |
